# finger-counter

An MVP finger counting app that uses your webcam and computer vision to detect one hand and estimate how many fingers are raised in real time.

## Requirements

- Python 3.10+
- A webcam

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python /home/runner/work/finger-counter/finger-counter/finger_counter.py
```

Press `q` to quit.
